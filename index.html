Web Maps

Hometown

This map was created as part of a MAP 672 assignment to learn how github, the leaftlet API, and other facets of development intersect to publicly disseminate web content using open source solutions.

Mural Map

The Lexington Mural Map was developed as a personal project to become more familiar with the city by finding and collecting information about murals. This map was created using QGIS for data development and Esri Story Maps for web dissemination.

Comp Plan Context

This Comprehsive Plan Context Map was created for the Lexington Planning Division to provide stakeholders with access to geographic data as related to the Comprehensive Plan. Data was provided by the city of Lexington and a Series Story Map was created to meet the goal.

Static Maps

The AT over Historic Map

This map of the Appalachian Trail over the 1957 Landforms of the United States by Erwin Raisz was created for a MAP 671 assignment. QGIS and the GDAL plug-in for georeferencing were leveraged.

Brewery Hexbins

This map of brewery density in in North America/Hawaii, created for MAP 671, uses the hexbid density method. The map and data creation were completed in QGIS using the MMQGIS toolset.

Bus Stop Density

This map of the city of San Diego, California demonstrates areas with varying degrees of access to bus stops per data extracted from Open Street Map and analyzed using QGIS as coursework for MAP 671.
